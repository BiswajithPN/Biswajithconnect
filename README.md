# 🌐 Biswajith Connect

A clean and modern personal portfolio website built using **HTML** and powered by **Firebase** for backend services.

🔗 Live Site: [Biswajith Connect](https://biswajithconnect.netlify.app/)

---

## ✨ About the Project

**Biswajith Connect** is a simple yet aesthetic personal portfolio website designed to showcase my profile, projects, and online presence.

The website is built using **pure HTML, CSS, and JavaScript**, with **Firebase** integrated for backend functionalities like data handling and hosting support.

---

## 🚀 Features

- ⚡ Lightweight and fast loading website  
- 📱 Fully responsive design for all devices  
- 🎨 Clean and minimal UI design  
- 🔥 Firebase integration for backend services  
- 🌍 Deployed on Netlify for global access  
- 🧭 Simple navigation and user-friendly layout  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript  
- Firebase (Database / Backend Services)  
- Netlify (Hosting)

---

## 🔥 Firebase Usage

This project uses Firebase for:

- Storing user/contact data (if applicable)
